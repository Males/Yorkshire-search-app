Yorkshire Products API Documentation

Welcome to the official documentation for the Yorkshire Products API.

Getting Started

To use the API, first obtain an API key by contacting tea@properbrew.co.uk.

Authentication

Authenticate every request by including the API key in a header: X-Yorkshire-Auth-Key.

Endpoints

  • GET /api/v1/pies – Returns all available pies.
  • POST /api/v1/brew – Starts a brew cycle for a cuppa.
  • GET /api/v1/whippets – Lists local whippets available for delivery.

Examples

GET https://yorkshireproducts.co.uk/api/v1/pies
Headers: { "X-Yorkshire-Auth-Key": "YOUR_API_KEY" }
